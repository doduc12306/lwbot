Commands List

Here you can find all the commands the bot has to offer, listed alphabetically.
Guild requires means the commands cannot be run in a DM.
All <arguments> are requried. All [arguments] are optional.
Commands listed in red are inaccessible.

Command Usage Description Aliases Category Requires
addemote addemote <emoji name> <image url> Adds an emote to the server ae, addemoji, createemote, createemoji Server Moderator - Manage Emojis - Guild
addtag addtag <name> <content> Adds a tag to the database tagadd, createtag Tags Bot Owner
announce announce <title> | <content> [| color=#hex no-subs] Sends an announcement to the server anno Server Moderator
ban ban <@user> [reason] Bans someone from the server Moderation Moderator - Ban Members
cowsay cowsay <text> root@lwbot-vps:~/lwbot-rewrite# cowsay Fun User
deletetag deletetag <tag name> Deletes a tag from the database removetag Tags Bot Owner
diagnose diagnose <command> See if a command is enabled/disabled System User
eco eco <add/remove/set/get> <amount> [user] Manage the bot's economy economy Economy Bot Admin
edittag edittag <tag name> <new content> Edit a tag etag Tags Bot Owner
eval eval <javascript> Executes arbitraty Javascript System Bot Owner
fight fight <@user> Fight a user! fite Fun User
